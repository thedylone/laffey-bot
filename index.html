use /valorant-watch or prefix command ?valorant-watch <name> <tag without #>. the bot will automatically retrieve your information and store in the database. you will be able to receive updates from the bot, view your info, and allow others to wait for you. note that the watch channel for the server must be set first if used in a server. if used in private messages, the bot will send updates to you directly instead.
